DSTIKE watch remote OS

  ### Requirements
 1. python packages:
     ```
     pip3 install -r requirements.txt
     ```

  2. Arial font (take a look also [here](https://askubuntu.com/a/651442/)):

      ```
     sudo apt-get install ttf-mscorefonts-installer
     sudo fc-cache
     fc-match Arial
     ```
